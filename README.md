## Featured Project: CRAFT SMS

<div style="background: linear-gradient(135deg, rgba(14, 165, 233, 0.15), rgba(6, 182, 212, 0.15)); border-radius: 12px; padding: 30px; margin: 30px 0; border: 1px solid rgba(14, 165, 233, 0.3);">

<h3 style="color: #0ea5e9; margin-top: 0;">
  <strong>CRAFT SMS</strong> — Production-Ready School Management SaaS
</h3>

**What it solves:**
Multi-tenant platform enabling schools to streamline student management, attendance tracking, academic records, and communication with a modern, intuitive interface.

**Architecture Highlights:**
- **Microservices** — Modular, independently deployable services
- **Event-Driven** — Real-time updates across tenants using event streams
- **Multi-Tenant** — Complete tenant isolation with shared infrastructure
- **API-First** — RESTful + GraphQL endpoints for flexibility
- **Cloud-Native** — Containerized, horizontally scalable
